# Sindhoor_AI
Building a Real-life JARVIS: The path to AI-Powered  Assistent
Developed a full-stack AI assistant featuring chat, code generation, resume analysis, and career guidance.
Integrated LLMs, vector databases (Pinecone), and machine learning models (XGBoost) to deliver personalized recommendations.
Built scalable REST APIs with Flask and implemented semantic search using embeddings for context-aware responses.
